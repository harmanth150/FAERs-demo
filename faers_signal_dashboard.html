Under the Hood

Core modules that support normalization, retrieval, pooling, and signal detection.

Drug Name Harmonization Maps brand names, generic names, dose strings, and product variants to one normalized FAERS query term.
Comparator Pooling Aggregates many comparator drugs into one reference market group for landscape-level disproportionality assessment.
Cached OpenFDA Retrieval Supports pagination, throttling, and local storage so users do not repeatedly fetch the same historical records.
Signal Detection Layer Builds 2×2 tables, applies corrections, confidence intervals, and FDR adjustment before ranking findings.
